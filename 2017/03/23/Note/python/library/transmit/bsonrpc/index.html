    文档

    python的bsonrpc库通过类装饰器的方式构造RPC服务

    • @service_class装饰RPC服务类
    • @request装饰提供RPC调用的函数
    • @rpc_request增加一个JSONRpc/BSONRpc对象作为参数
    • @notification用于通知，有个字符串参数
