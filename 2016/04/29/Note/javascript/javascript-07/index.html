    显示缩略语列表

    使用JavaScript把abbr标签的title属性集中在一起显示，创建的显示列表如下图的格式

    具体步骤如下

    • 遍历这份文档中的所有abbr元素
    • 保存每个abbr元素的title属性
    • 保存每个abbr元素包含的文本
    • 创建一个列表元素(dl)
    • 遍历刚才保存的title属性和abbr元素的文本
    • 创建一个标题元素(dt)
    • 创建一个描述元素(dd)
    • 把abbr文本属性插入到标题元素(dt)
    • 把title属性插入到描述元素(dd)
    • 把dt和dd追加到dl上
    • 把dl追加到body上
