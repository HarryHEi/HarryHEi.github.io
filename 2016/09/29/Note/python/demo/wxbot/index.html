    爬虫加微信接口

    • 功能是微信收到消息之后，request爬百度对应的图片，然后回过去

    • 项目地址mywxbot

    • 发送图片的时候如果设置为中文名称就会出错

    • 首先用微信接受字符串，注意要把unicode编码转成str

    • 百度的图片资源可能会403

    • 图片名是用的随机字符串，也可以用名称的md5值，不过会出现图片重复，这样可以避免重复下载

    • 下载图片用的是request文档的推荐，也可以用wget，不过之前试的时候经常403

    • 爬到的图片不是高清的

    • 总之，也算基本完成预期吧
