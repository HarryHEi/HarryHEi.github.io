    复选框

    最基本的应用是对复选框的全选、反选操作
    主要通过复选框选诉的checked属性来设置
    当checked属性为true时，说明被选中，反之没有

    首先建立一个表单，有一些复选框和一个全选复选框
    当勾选”全选”复选框时，全部勾选
    取消勾选”全部”复选框时，全部不勾选
    按下提交按钮时，获取到checked属性为true的复选框的值，然后都打印出来

    这里用到了prop方法更改checked属性，因为jQuery的库的兼容问题，如果使用attr会出现第二次不能勾选的情况

    遍历每个对象使用了each()方法
