    简述

    Django是python下的另一款服务器网络框架，属于重量级框架
    MVC架构指的是：模型(model)，视图(views)，控制(controller)
    由于Django自行处理控制部分即C，所以Django也被称为MTV框架，这里T指的是模板(Template)
