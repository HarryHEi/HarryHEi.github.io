    创建一个class的主要步骤如下：

    • 首先检查Empty中是否有__metaclass__(python2)
    • 如果有，使用这个metaclass创建一个名为Empty的class对象。
    • 如果找不到__metaclass__，在模块级别查找，如果依然找不到，在Base中查找。

    注意的是，__metaclass__不会被继承。
