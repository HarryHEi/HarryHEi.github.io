    CMD、ENTRYPOINT

    CMD是容器启动时默认执行命令；
    ENTRYPOINT是镜像的默认入口命令；
    每个Dockerfile两者都只能有一个，后面的会代替前面的，CMD可以是提供给ENTRYPOINT的默认参数。
